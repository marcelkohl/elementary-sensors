# elementary-vala-gtk-sensors
An attempt to make visible temperature from sensors cli

## build / compile
[build system](https://docs.elementary.io/develop/writing-apps/untitled/the-build-system)
```
meson build --prefix=/usr
cd build
ninja
ninja install
```
